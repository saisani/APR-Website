AWS Sagemaker

The following instructions will show you how to install donkeycar on an AWS SageMaker Notebook instance.

The Notebook instance is typically used for experimenting and preparing your data and model for training. The convention is then to distribute the training on a separate cluster of training instances.

This, however, requires you to adapt your training script (and model) to work in a distributed manner, preferably using the SageMaker Python SDK, which is not currently available in Donkey.

That said, it is still possible to install and train your model on the Notebook instance. It will allow you to train your Keras model using a beefy instance type with a large GPU, and, perhaps most importantly, shut it down when finished so that you only pay for what you use.

Create a Notebook instance

f you havn't already, log in to your AWS Console and create a new AWS SageMaker Notebook instance:

https://docs.aws.amazon.com/sagemaker/latest/dg/how-it-works-notebooks-instances.html#howitworks-create-ws

Suggest using a ml.p2.xlarge instance type. You can find a list of available types here:

https://aws.amazon.com/sagemaker/pricing/instance-types/
